# 🎓 Maths-2: Interactive Learning Hub

[![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)](https://vitejs.dev/)
[![Neo-Brutalism](https://img.shields.io/badge/Design-Neo--Brutalism-FFD600?style=for-the-badge)](https://www.uxdesign.cc/)

**Maths-2 Compiler** is a specialized interactive learning platform designed to help Computer Science students master Engineering Mathematics. Moving away from traditional "reading-only" methods, this application utilizes **live simulations, visualizers, and interactive playgrounds** to make abstract mathematical concepts tangible.

---

## 🚀 Key Features

- **Interactive Playgrounds:** Every topic comes with custom-built tools (Calculus visualizers, Probability simulators, Z-Score calculators) to "play" with the math before solving problems.
- **Syllabus-Guided Path:** A strictly sequenced 13-unit curriculum that takes students from Foundations of Probability to advanced Statistical Inference.
- **Human-Centric Design:** Built with a **Neo-Brutalism aesthetic** and hand-written typography to reduce "exam anxiety" and make learning feel more approachable.
- **Professional Lectures:** Integrated failover-ready video resources for every single topic.
- **Mastery Tracking:** Real-time progress monitoring to ensure students are ready for their end-semester exams.

---

## 🛠️ Tech Stack

- **Frontend:** React.js (Hooks, Context API)
- **Styling:** Vanilla CSS (Neo-Brutalism Framework)
- **Mathematical Rendering:** KaTeX & React-Latex-Next
- **Build Tool:** Vite
- **Icons:** Emoji-based System for maximum compatibility and fun

---

## 📈 Curriculum Overview

The hub covers the complete "Maths for CS II" syllabus:
1. **Probability Foundations** (Axioms, Set Logic)
2. **Conditional Logic** (Bayes' Theorem)
3. **Counting Principles** (Combinatorics)
4. **Random Variables** (Discrete & Continuous)
5. **Statistical Inference** (Hypothesis Testing, Z-Tests)
6. **Limit Theorems** (CLT, LLN)

## 🔧 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ManojAhire/Maths_Exam_Prep.git
   cd Maths_Exam_Prep
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

---

## 💡 Why I Built This

I developed this project to solve a specific pain point: the gap between theoretical math and practical intuition. By building interactive visualizers, I wanted to show that complex concepts like the **Central Limit Theorem** or **Bayesian Updates** can be intuitive when you can interact with the variables in real-time.
